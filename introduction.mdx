## Overview

Welcome to Yogesh Agrawal's documentation platform. You can organize, manage, and share all your project documentation in one centralized space. This platform helps you create structured docs, collaborate with teams, and keep everything up to date with version control and search capabilities.

Use this space to document APIs, user guides, internal wikis, or any project-related knowledge. Start by customizing these pages to match your specific needs.

## Quick Start

Get up and running in minutes with these essential steps.

<Steps>
  <Step title="Create Account" icon="user-plus">
    Sign up at `https://docs.example.com/signup` using your email or GitHub account.
  </Step>
  <Step title="Set Up Space" icon="settings">
    Create a new documentation space and configure your brand color (`#3B82F6`).
  </Step>
  <Step title="Add First Page" icon="plus">
    Click "New Page" and start writing with markdown or import from GitHub.
  </Step>
  <Step title="Publish" icon="globe">
    Preview changes and publish your docs for team access.
  </Step>
</Steps>

## Navigation Guide

Navigate your documentation space efficiently using these options.

<Tabs>
  <Tab title="Sidebar" icon="menu">
    Use the left sidebar to browse pages, sections, and search results. Pin frequently used pages for quick access.
  </Tab>
  <Tab title="Search" icon="search">
    Press `<kbd>Ctrl</kbd>+<kbd>K</kbd>` (or `<kbd>Cmd</kbd>+<kbd>K</kbd>` on macOS) to open global search across all docs.
  </Tab>
  <Tab title="Breadcrumbs" icon="chevron-right">
    Follow the path at the top of each page to jump between parent sections quickly.
  </Tab>
</Tabs>
