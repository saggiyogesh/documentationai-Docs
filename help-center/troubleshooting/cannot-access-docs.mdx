---
title: Cannot Access Documentation
description: Steps to resolve access issues.
---

## Symptoms

- Unable to load docs
- See a 403 or 404 error

## Causes

- Not logged in
- No access rights for project

## Solution

<Steps>
  <Step title="Check login status">
    ````
    Go to https://docs.yogeshagrawal.com and make sure you are signed in.
    ````
  </Step>
  <Step title="Verify permissions">
    ````
    Ask the project owner to grant you access if needed.
    ````
  </Step>
</Steps>
