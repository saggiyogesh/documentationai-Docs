<Update label="2024-10-15" description="v1.2.0" tags={["feature", "bugfix"]}>

## New Features

- Added full-text search across all documentation pages with syntax highlighting support
- Introduced dark mode toggle with automatic system preference detection
- New version comparison tool for changelogs and API endpoints

## Improvements

- Enhanced page load performance by `>30%` through code splitting and lazy loading
- Improved mobile responsiveness for sidebar navigation

## Bug Fixes

- Fixed collapsible sections not persisting state on page refresh
- Resolved anchor link scrolling issues on long pages
- Corrected table rendering glitches in certain browsers

</Update>

## Migration Guides

<ExpandableGroup>
  <Expandable title="Migrate from v0.x to v1.0.0" default-open="true">

Follow these steps to update your documentation site:

<Steps>
  <Step title="Backup Config" icon="database">

    Save your current configuration:

````bash
cp config.yaml config.yaml.backup
````

  </Step>
  <Step title="Update Dependencies" icon="package">

    Install the latest version:

````bash
npm install yogesh-agrawal-docs@latest
````

  </Step>
  <Step title="Convert Shortcodes" icon="code">

    Replace legacy shortcodes with MDX components. For example:

    <CodeGroup tabs="Before,After">
````jsx
<!-- Before -->
<Callout type="info">Note</Callout>
````
````jsx
<!-- After -->
<Callout kind="info">Note</Callout>
````
    </CodeGroup>

  </Step>
  <Step title="Test Locally" icon="play">

    Run development server:

````bash
npm run dev
````

  </Step>
</Steps>

  </Expandable>

  <Expandable title="Migrate from v1.0 to v1.1+">

    Minimal changes required. Update your theme config to support new icons:

````json
{
  "theme": {
    "primaryColor": "#3B82F6",
    "icons": "lucide"
  }
}
````

  </Expandable>
</ExpandableGroup>
